# WebTantu | Architectural Digital Solutions

This is the fully responsive, animated front-end project for the WebTantu homepage, built with React, Vite, Tailwind CSS v4 (using the exact "Editorial Emerald" Stitch design system), and Framer Motion.

## Prerequisites
Before running this project on any new device or code editor, you must have the following installed:
- **Node.js**: Download and install it from [nodejs.org](https://nodejs.org/). (We recommend the LTS version).

## Setup & Running the Project

### Option 1: One-Click Setup (Windows)
We've included a `setup.bat` file for Windows users. Simply double-click `setup.bat` in the project folder to automatically install all dependencies and start the local server.

### Option 2: Manual Setup (Any OS)
If you prefer using your terminal or are on macOS/Linux, follow these steps:

1. Open a terminal or your code editor (like VS Code) and navigate to the project folder.
2. Install the necessary dependencies by running:
   ```bash
   npm install
   ```
3. Start the local development server by running:
   ```bash
   npm run dev
   ```
4. The terminal will display a local host link (typically `http://localhost:5173/`). Ctrl+Click (or Cmd+Click) the link to view the website in your browser!

## Project Stack
- **React + Vite**: High-performance frontend framework and build tooling.
- **Tailwind CSS v4**: Utility-first styling tailored to specific design tokens via `src/index.css`.
- **Framer Motion**: Expert-level, physics-based scroll reveals and interactive animations.
